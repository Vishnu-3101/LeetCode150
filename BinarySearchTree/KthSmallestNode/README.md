Given the root of a binary search tree, and an integer k, return the kth smallest value (1-indexed) of all the values of the nodes in the tree.<br/>

 

### Example 1:


**Input:** root = [3,1,4,null,2], k = 1<br/>
**Output:** 1<br/>
