[Problem](https://leetcode.com/problems/longest-palindromic-substring/description/)<br/><br/>

Given a string s, return the longest palindromic substring in s.<br/>

### Example 1:

**Input**: s = "babad"<br/>
**Output**: "bab"<br/>
**Explanation**: "aba" is also a valid answer.<br/>

### Example 2:

**Input**: s = "cbbd"<br/>
**Output**: "bb"<br/>
