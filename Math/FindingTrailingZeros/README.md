Given an integer n, return the number of trailing zeroes in n!.<br/>

Note that n! = n * (n - 1) * (n - 2) * ... * 3 * 2 * 1.<br/>

 

### Example 1:

**Input**: n = 3<br/>
**Output**: 0<br/>
**Explanation**: 3! = 6, no trailing zero.<br/>

### Example 2:

**Input**: n = 5<br/>
**Output**: 1<br/>
**Explanation**: 5! = 120, one trailing zero.<br/>
